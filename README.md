# Poly-Golf
A web based multiplayer golf game made in unity

## MVP Features
- Single player golf 
- Multiplayer golf 
- Level editor 

### Single Player

- Isometric
- Tile based
- Rotate camera 
- Arrow UI to show direction and Power


### Multiplayer

- Colour coded 
- PvP collision enabled


### Level editor

- Drag and Drop (N x N fixed size grid)
- Save and Export config
- Upload config to play with friends
